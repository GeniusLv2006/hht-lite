3. 数据收集与隐私保护

3.1 数据收集范围

  • OpenID、IP 地址：由开发者收集，用于调用官方 API 生成二维码及安全审计，作为服务器安全日志留存，保留期不超过 30 天。
  • 服务器访问日志：由开发者收集，用于安全审计与故障排查，保留期不超过 30 天。
  • 匿名统计数据：由 Cloudflare 收集，用于网站访问量统计分析，依 Cloudflare 隐私政策独立处理。

3.2 Cloudflare Web Analytics

本服务集成 Cloudflare Web Analytics，仅收集匿名聚合统计信息（如页面访问量、访问来源地区等），不追踪个人用户身份，不使用 Cookie。相关数据由 Cloudflare, Inc. 依其隐私政策独立处理与存储，开发者无法获取任何个人身份信息。

3.3 数据使用原则

  • 开发者持有的数据仅用于服务安全维护目的，不用于任何商业目的，不向无关第三方出售、出租或披露。
  • 数据传输全程采用 HTTPS 加密协议保护。
  • 日志数据将定期清理，保留期不超过 30 天。

3.4 安全事件响应

如发生数据安全事件，开发者将在知悉后尽合理努力于本服务显著位置公开披露相关情况及已采取的补救措施。

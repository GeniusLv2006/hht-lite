服务说明与使用协议
生效日期：2026 年 4 月 11 日（本次修订）
1. 定义
  • "本服务"：指通过 huihutong.xjtlu.uk 提供的宿舍门禁二维码生成功能及全部相关内容。
  • "开发者"：指本服务的独立开发者及运营者。
  • "用户"：指访问或使用本服务的任何个人。
  • "OpenID"：指用户在慧湖通系统中的唯一身份标识符，同时作为宿舍门禁二维码的生成凭据。
  • "账户信息"：指用户通过慧湖通官方系统身份验证后，由官方 API 自动返回的账户显示名称及系统标识符，本服务将其用于管理后台辅助识别。
2. 服务性质与独立声明

本服务系开发者独立开发与运营的非营利性个人项目，其功能为调用慧湖通官方 API 生成宿舍门禁二维码，旨在提升用户使用体验。

本服务与以下机构不存在任何形式的隶属、授权、合作或背书关系：西交利物浦大学（XJTLU）、苏州独墅湖科教发展有限公司。

商标声明："慧湖通"为苏州独墅湖科教发展有限公司的注册商标。本服务对该名称的使用仅出于描述性目的，以说明本服务所对接的官方系统，不暗示任何形式的关联或授权。

本服务以非正式方式发布，不构成上述任何机构的官方产品或服务。如需使用官方服务，请通过慧湖通官方小程序操作。

3. 数据收集与隐私保护

3.1 数据收集范围

  • OpenID、IP 地址及 IP 归属地：由开发者收集，用于调用官方 API 生成二维码及安全审计，作为服务器安全日志留存，保留期不超过 30 天。其中，IP 归属地信息通过第三方服务 ip.sb 的 GeoIP API 查询获得，查询过程中该服务将接收用户 IP 地址，相关数据处理依 ip.sb 隐私政策执行。
  • 账户信息：用户身份验证成功后，由慧湖通官方 API 自动返回的账户显示名称及系统标识符，本服务将其存储于服务端用于后台管理辅助，可根据请求删除（见第 11 条）。
  • 服务器访问日志：由开发者收集，用于安全审计与故障排查，保留期不超过 30 天。
  • 匿名统计数据：由 Cloudflare 收集，用于网站访问量统计分析，依 Cloudflare 隐私政策独立处理。

3.2 Cloudflare Web Analytics

本服务集成 Cloudflare Web Analytics，仅收集匿名聚合统计信息（如页面访问量、访问来源地区等），不追踪个人用户身份，不使用 Cookie。相关数据由 Cloudflare, Inc. 依其隐私政策独立处理与存储，开发者无法获取任何个人身份信息。

3.3 数据使用原则

  • 开发者持有的数据仅用于服务安全维护目的，不用于任何商业目的，不向无关第三方出售、出租或披露。
  • 数据传输全程采用 HTTPS 加密协议保护。
  • 日志数据将定期清理，保留期不超过 30 天。

3.4 安全事件响应

如发生数据安全事件，开发者将在知悉后尽合理努力于本服务显著位置公开披露相关情况及已采取的补救措施。

3.5 客户端本地存储（localStorage）

本服务在用户设备浏览器的 localStorage 中存储以下信息，用于维持功能与偏好设置：OpenID（用户填写后本地保存）、登录凭据（satoken，有效期与官方 API 一致）、账户显示名称（用于页面问候显示）、用户偏好设置（如 QR 码容错等级）。上述数据仅存于用户本人设备，不通过本服务发送至任何第三方，用户可随时通过浏览器设置清除。

7. 安全提示

OpenID 是您在慧湖通系统中的唯一身份标识，同时也是生成宿舍门禁二维码的直接访问凭据，其安全性与您的人身及财产安全直接相关。

  • OpenID 一旦泄露，他人即可利用其生成您的门禁二维码，对宿舍物理安全构成直接威胁。
  • 请务必妥善保管，切勿以截图、转发或任何其他形式向他人披露。
  • 如怀疑 OpenID 已遭泄露或被滥用，请立即停止使用本服务并联系开发者。
  • 本服务在服务端保存了您的 OpenID 与账户信息的关联记录，仅用于安全审计，不对外披露。如发生数据安全事件，开发者将按第 3.4 条约定及时处理，并可根据请求删除相关数据（见第 11 条）。

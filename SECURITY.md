# Security Policy

## Maintenance status

hht-lite has discontinued active maintenance. The repository remains available, but there is no commitment to provide security updates, vulnerability triage, compatibility fixes, or a response within any timeframe.

Public demo availability does not indicate that the software is secure or supported. Anyone operating a fork or self-hosted instance is responsible for reviewing the code, dependencies, configuration, data handling, and upstream API behavior, and for applying necessary mitigations.

## Reporting a vulnerability

Do not disclose unpatched vulnerabilities, credentials, OpenIDs, access logs, or other sensitive data in a public issue. Use GitHub's private vulnerability reporting feature for this repository instead.

Because active maintenance has ended, private reports may not receive a response or result in a fix. If you cannot safely operate the affected code without a maintainer-provided patch, stop using the software or maintain a secured fork.
